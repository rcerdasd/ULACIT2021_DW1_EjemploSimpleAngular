Ejemplo de aplicacion con Angular

El archivo home.component.ts en esta aplicacion establece un componente con Angular que lee un archivo HTML en la misma carpeta que contiene este parrafo. Debido a que el componente existe ya, lo agregé facilmente las otras secciones de este proyecto únicamente llamando al id app-home. En la página home además, llamé los otros componentes con solo llamarlos por su respectivo id.
